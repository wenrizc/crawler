### 1. 项目与新技术栈

*   **Agent**：
    *   AI Agent的架构设计（如ReAct模式）。

### 2. 框架与源码

    *   Spring IOC（控制反转）和 AOP（面向切面）的原理。
    *   Spring Bean 的生命周期。
    *   Spring 的循环依赖是如何解决的？

### 3. Java 基础与并发 (JUC & JVM)
*   **Java 基础**：
    *   集合框架（HashMap 的底层结构、扩容机制、ConcurrentHashMap）。
    *   面向对象特性（多态、继承等）。
*   **JUC (Java Util Concurrent)**：
    *   线程池的参数配置及工作原理。
    *   锁机制（Synchronized vs ReentrantLock）。
    *   CAS (Compare And Swap) 及 AQS (AbstractQueuedSynchronizer) 原理。
    *   JUC 包下的并发集合与工具类。
*   **JVM**：
    *   JVM 内存模型（堆、栈、方法区等）。
    *   垃圾回收算法（GC Root可达性分析、标记清除/整理、复制算法）。
    *   垃圾收集器（CMS, G1, ZGC 等）。
    *   类加载机制（双亲委派模型）。
    *   JVM 调优经验与参数设置。

    *   乐观锁与悲观锁的实现。
*   **事务**：
    *   ACID 特性。
    *   Spring 事务失效的场景。
*   **隔离机制 & MVCC**：
    *   MySQL 的四种隔离级别及对应解决的问题（脏读、幻读、不可重复读）。

*   **Read View**：
    *   Read View 的结构及其在 MVCC 中如何判断可见性。
*   **日志**：
    *   Redo Log（重做日志）与 Binlog（归档日志）的区别与作用。
    *   Undo Log（回滚日志）的作用。
