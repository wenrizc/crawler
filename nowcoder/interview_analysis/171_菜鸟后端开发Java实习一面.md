**一、 聊实习和项目**
1. 聊一下模糊查询（相关实现或细节）。
2. 缓存一致性是如何处理的？
3. 过期时间是如何设置的？
4. 查询或者存储的数据量是多少？
5. 缓存命中率大概有多高？
6. 项目中有没有什么难点？是怎么解决的？

1. 有哪些AI相关的经历？
2. 为什么搞Java（为什么选择Java方向）？
3. 有看书或者技术博客的习惯吗？

**三、 一些常规八股（技术基础）**

2. Spring事务的传播属性有哪些？
3. ThreadLocal的原理是什么？使用场景有哪些？
4. 介绍一下数据库的锁。
5. MySQL的悲观锁用什么关键字？
6. 举一些Linux指令的例子。
