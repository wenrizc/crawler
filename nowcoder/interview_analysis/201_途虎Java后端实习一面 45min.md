根据提供的面试记录，以下是解析出的所有面试问题：

**Java基础**
1. 如何定义一个线程
2. 线程池的核心参数
3. synchronized 和 reentrantLock区别
4. 公平锁和非公平锁区别

**JVM相关**
1. 简单讲一下JVM
2. 基于JVM模型，讲一下堆和栈的区别
3. 为什么要分堆和栈两种来管理内存（只有堆或者只有栈的话不行吗）

**Spring相关**
1. spring boot 它比原生的 spring 有哪些不同呢？或者说它额外做了哪些事情？
2. Spring Boot 自动装配
3. Spring定义一个Bean有哪些方法（@component）
4. 展开讲讲AOP
5. 知道静态代理吗
6. 要实现AOP，在开发层要做什么，用伪代码写一写

**数据库相关**
1. 表a和表b有多对多的关系，怎么管理
2. select id from table_a where id = xxx 和 select * from table_b where id = xxx ，这两个查询语句有什么区别
