根据提供的面试记录，以下是解析出的所有面试问题，已按主题分类整理：

**1. 算法与排序**
*   冒泡排序和选择排序的时间复杂度是多少？
*   如何寻找第 k 大的数字？（提示使用堆排序）
    *   应该使用大根堆还是小根堆？
    *   堆的构建过程是怎样的？

**2. Java 与数据结构**
*   看你技术栈是 Java，讲讲 Java 里用到了哪些数据结构？
    *   （针对 ArrayList 和 LinkedList）两者的区别是什么？各自的优势是什么？
    *   （针对 HashMap）底层是数组，数组索引怎么确认？是通过哈希计算得到的吗？
    *   常见的哈希算法了解哪些？解决哈希冲突的方法呢？
    *   红黑树了解多少？

**3. 计算机网络（TCP）**
*   TCP 的 `TIME_WAIT` 状态是出现在主动断开连接的那一方还是被动的那一方？
*   `TIME_WAIT` 状态的作用是什么？
*   你提到 TCP 超时重传，这个机制的底层实现原理有了解吗？

**4. 操作系统**
*   操作系统中虚拟内存和物理内存的区别是什么？
*   内存置换算法了解哪些？

**5. 项目经验**
*   项目中最大的困难是什么？
*   是如何解决的？
*   项目有哪些不足？

**6. 其他与综合**
*   除了写代码，你还有其他什么兴趣爱好？
*   你有什么问题想问我们吗？（反问环节）
