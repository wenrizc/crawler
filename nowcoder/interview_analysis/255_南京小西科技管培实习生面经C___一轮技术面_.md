基于提供的面试记录，以下是解析出的所有面试问题。为了方便阅读，我按照知识模块对问题进行了分类：

### 1. 基础介绍
*   **请做一个自我介绍。**

### 2. 计算机网络（TCP/IP）
*   **请介绍一下 TCP。**
    *   TCP 在内核中是如何存储半连接队列的？（涉及 TCB）
    *   ACK 机制相关：是否了解 SACK（选择性确认）？
    *   TCP 首部（20字节格式）包含哪些字段？

### 3. 操作系统与并发编程
*   **关于任务队列的问题。**
*   **条件变量与锁的原理及使用。**
*   **什么是虚假唤醒？**（以及在 C 语言中如何使用 `while` 循环，C++ 中如何使用函数对象/谓词来检测/避免？）

### 4. 项目细节与高并发
*   **请讲解一下 EPOLL + 线程池的高并发处理思路。**
*   **LRU（最近最少使用）缓存是如何实现的？**（涉及双向链表 + 哈希 map）
*   **请解释 mmap、sendfile、splice 的概念和作用。**（涉及零拷贝技术）

### 5. 性能优化与内存管理
*   **你是如何进行性能优化的？**
*   **什么是内存泄漏？**（以及如何使用工具如 Valgrind 进行检测？）
*   **如何进行内存定位？**（排查内存问题的方法）

### 6. 开发工具与调试（GDB/Linux）
*   **介绍一下 GDB 的使用。**（涉及线程、栈、断点、观察点等方面）
*   **常用的 Linux 命令有哪些？**（具体涉及 ss / netstat 查看 TCP 连接状态）
*   **如何利用 core 文件进行错误定位？**
*   **关于 GDB 或 VS Code 插件的使用。**
*   **遇到段错误（Segmentation Fault）时如何定位和解决？**
