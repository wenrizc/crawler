1. Docker沙箱是每次提交代码时候都会创建新的吗，还是会复用？
2. 判题逻辑是怎么样的，最后样例输出是直接读取Docker沙箱的输出吗，还是说Docker沙箱里面有自己的业务？

5. 排行榜是根据什么来排名次的？

### 三、 Redis
1. 说说Redis一些常见命令。
