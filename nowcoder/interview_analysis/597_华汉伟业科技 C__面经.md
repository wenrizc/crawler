根据提供的面试记录，以下是解析出的所有面试问题：

1.  **结构体和类的区别是什么？**
2.  **结构体是否可以继承类？类是否可以继承结构体？结构体和结构体之间、类和类之间是否可以继承？**
3.  **在单继承的情况下，构造函数和析构函数的调用顺序是怎样的？（注：需包含成员变量的初始化顺序）**
4.  **纯虚函数怎么写？**
5.  **一个类的多个实例之间如何进行数据共享？**
6.  **请解释一下希尔排序。**
7.  **希尔排序是否稳定？什么是“稳定”的概念？**
8.  **请手写一下 string 的构造函数和拷贝构造函数。**
